Click and drag this element to get it to scroll



This is done by C# methods which can be reused anywhere in the project. When this element is clicked, it gets focused and the C# script, DragScroller, stores the location of the mouse. Because it's focused, onmousemove also occurs. Mouse up might not occur on this element though, so the mousemove function looks out for a mouse up. When one occurs, it simply unfocuses this element.

Note the use of text-clip on the scrollers style. Optional of course, but this disables an optimisation which makes text appear to get squished otherwise.
